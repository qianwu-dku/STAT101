Data, Variable, Model

What is Statistics?

Statistics is the science that deals with the collection, analysis, and presentation of data, which can then be used as a basis for inference and induction.

Data

  • Data refer to any collection of numbers, characters, images, or other items that provide information about something.
  • All the data collected for a particular study are referred to as the data set for the study.

Element, Variable, and Observation

  • Elements are the entities on which data are collected.
  • Variables are characteristics recorded about each element.
  • Observation is the set of measurements obtained for a particular element.
    • A data set with n elements contains n observations

Example: Walmart Sales

Discuss in groups:

  • What is an element in this table?
  • How many variables are there?
  • What is an observation in this table?

What is a Model?

  • Models are summaries and simplifications of data that help our understanding in many ways.
  • It’s a simplification of reality that gives us information, even though it doesn’t represent reality exactly.
  • “All models are wrong, but some are useful.”

Example: Study Hours and Exam Score

# Simple curved data
library(ggplot2)
library(dplyr)
library(tidyr)

# Create a dataset for the line
set.seed(123)
x <- 1:20
y <- 10 + 2*x + 0.2*x^2 + rnorm(20, 0, 2)
simple_data <- data.frame(x, y)

line_data <- data.frame(
  x = simple_data$x,
  y = predict(lm(y ~ x, data = simple_data))
)

# Combine data for legend
plot_data <- bind_rows(
  simple_data %>% mutate(type = "Data Points"),
  line_data %>% mutate(type = "Linear Model")
)

ggplot(plot_data, aes(x = x, y = y, color = type)) +
  geom_point(data = filter(plot_data, type == "Data Points"), size = 2) +
  geom_line(data = filter(plot_data, type == "Linear Model"), linewidth = 1) +
  scale_color_manual(values = c("Data Points" = "red", "Linear Model" = "blue")) +
  labs(title = "",
       x = "Study Hours", y = "Exam Score",
       color = NULL) +
  theme_minimal() +
  theme(axis.title = element_text(size = 14),
        legend.position = "top")

Qualitative and Quantitative Data

  • Qualitative variables tell us what group or category each individual belongs to.
    • E.g., eye color, education level, ZIP code
  • Quantitative variables contain measured numerical values with measurement units.
    • E.g., age, income, apartment size

Display Qualitative Data

Frequency tables

  • A frequency table lists the number of cases in each category along with the category name.

  • A relative frequency table displays percentages (or proportions) rather than the counts in each category.

Example: Class grade

Bar chart

  • On one axis (usually, horizontal), we specify the class names.
  • A frequency or relative frequency scale can be used for the other axis (usually, vertical).

Example: Class grade

Pie chart

  • Each segment of the pie represents the relative frequency of one category.
  • Use a pie chart to compare the relative sizes of all possible categories.

Example: Class grade

Display Quantitative Data

Frequency Tables

  • We can still use frequency and relative frequency tables to display quantitative data, techniques are similar to those with qualitative data.
  • However, for quantitative data, you need to determine the number of classes.
    • Check here if you are interested in how to determine the number of classes (and class width, accordingly).

Histogram

  • A histogram slices up possible values into bins, then the bars show the frequency or relative frequency for each bin.

Example: iPad sale

Relationships between Qualitative Variables

  • Are people with blue eyes more likely to be left-handed?
  • Is there a relationship between Internet use and race?
  • Are dog people also tend to be cat people?

Contingency Table

A contingency table displays the multivariate frequency or relative frequency distribution of categorical variables.

  • Marginal Total: The numbers of the males (52), females (48), and right- (87) and left-handed (13) individuals.

  • Grand Total: the total number of individuals represented in the contingency table (100).
